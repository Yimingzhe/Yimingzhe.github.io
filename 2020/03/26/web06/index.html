对基本开发环境在此做简单的记录。

要对新电脑的开发环境进行配置，在此仅作一个简单的记录。

（一） IDE的下载

选择VSCode或者Intellij idea。

（二）Node.js

​ nodejs官网 默认配置，查看版本：node –version npm –version

​ nvm: node管理工具，解决各版本node不兼容问题

​ nvm –version

​ nvm –help

​ nvm –ls 查看版本

​ nvm install v.11.1.1

​ 淘宝npm镜像

​ http://npm.taobao.org/

​ cnpm命令下载 cnpm install

（三）调试环境: Chrome控制台与Chrome中Vue的插件

<https://chrome-extension-downloader.com/>

​ 下载码：nhdogjmejiglipccpnnnanhbledajbpd

​ 如果可以访问Google，直接在应用商店中查找vue即可。

​ 查看扩展程序，将安装的vue插件拖拽到浏览器上面。

（四）工程环境

​ 安装Vue Cli, “npm i -g vue-cli” 新版本（目前Vue CLI3.0）：npm install -g @vue/cli

在此简单回顾Vue CLI创建新项目的步骤：

方式一： 通过命令行创建

cd进入将新建项目的目录，再通过vue create hello-world创建名为”hello-world“的项目，名称用小写字母，中间用-或_间隔；

选择manually select features，上下箭头选择，按空格来选中。假设上一步加选了router, vuex, css pre-processors, 以下是基本的一些设置: history mode for router: yes; css pre-processors: Sass/SCSS; Pick a linter/formatter config: 一般选standard，选airbnb也可。lint features: lint on save; preset: 一般选no，不用作未来预设。再根据提示即可。

方式二： 通过vue ui创建

直接在命令行输入vue ui，在界面中进行一些设置。create创建新项目, 包管理器选择npm, 和方式一相同，手动配置项目，在界面上选择需要的插件，对插件进行设置。运行即可。
