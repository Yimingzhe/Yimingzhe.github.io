Hexo常用命令

  1. 新建一篇文章： hexo new typeTitleHere 或 hexo n typeTitleHere

  1. 启动服务观看效果

    hexo server 或 hexo s, 即可在浏览器输入 localhost: 4000进行本地查看。

  1. 重新部署更新

    hexo clean(清理本地缓存) hexo generate(生成静态文件，简写为hexo g) hexo deploy(部署，简写为hexo d)

markdown常用基本语法

标题

想要生成h1到h6的标题，只需要在标题文字前增加对应数量的“#”，最后一个”#”之后加一个空格。

强调

前后各用两个星号包裹起来的文字会被加粗强调。

代码

可以使用三个`(键盘上左上角数字1左边的符号)包裹代码，并指明语言
