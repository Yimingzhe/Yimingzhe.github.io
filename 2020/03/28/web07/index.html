VUE学习简单实例

发表于 2020-03-28 | 分类于 Web Technologies | 阅读次数：
本文字数： 976 | 阅读时长 ≈ 1 分钟
一个简单的音乐播放器，Vue初级学习实例。并记录Github Page的基本使用方法。

Music Player

参照教程写了简单的音乐播放器，其中API来自网易云音乐。

Github Page

GitHub Pages是展现demo的很好方式，可以快速部署静态网页。在此实例中运用此进行展示，效果可以点击Music Player观看。

以音乐播放器的步骤为例，首先New repository创建MusicPlayer，创建完成后在设置中找到 Github Pages, 将Source改为master branch，这样就可以在该栏中看到 “Your site is published at https://yimingzhe.github.io/musicPlayer/“ 的提示。这个网址就可以展示项目了。

接下来使用git将本地仓库的内容上传到远程。

（1）在本地项目目录中使用git bash here, 执行git init，将创建一个隐藏文件。

（2）使用“git add .”命令将全部文件添加到暂存区。

（3）使用git commit -m “first upload”将文件提交到仓库，引号内表示提交的说明。

（4）需要关联到远程库，通过git remote add origin https://github.com/Yimingzhe/musicPlayer.git进行关联。地址是远程仓库的地址，加后缀名.git。

（5）如果远程库已经有内容，还需要将远程库和本地同步合并。git pull –rebase origin master

（6）将本地库中的内容推送到远程。git push -u origin master

补充：git常用命令

删除文件：git rm filename。这会直接将文件从当前目录删除。如果想保存在工作目录，只是从git上删除，可以使用git rm –cached filename

查看修改：git diff

查看和切换分支：git branch查看所有分支；git branch branchName创建新分支；git checkout branchName切换到某一分支。

合并分支：假设将dev分支合并到master上，需要：git checkout master（切换到master分支上）；git merge dev。可以用git branch -d dev删除多余分支。
